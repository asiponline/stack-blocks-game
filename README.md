# Stack Blocks Game

一个轻量的 Web 端叠方块互动小游戏，使用原生 HTML、CSS 和 JavaScript 开发。

## 在线部署

本项目可直接部署到 GitHub Pages：

1. 将代码推送到 GitHub 仓库。
2. 进入仓库 `Settings` → `Pages`。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub Pages 构建完成。

## 本地运行

直接用浏览器打开 `index.html` 即可。

也可以在当前目录启动任意静态服务器，例如：

```bash
npx serve .
```

## 操作方式

- 鼠标点击画布：落下方块
- 移动端点击“落下方块”按钮：落下方块
- 键盘：空格、回车、方向下键
